# 📝 Solution Templates

Copy the appropriate template when starting a new solution — or just use
`scripts/new.py` to have it done automatically.

## Manual usage

```bash
# Python — e.g. problem 0121 in trees/
cp templates/template.py trees/0121-best-time-to-buy-and-sell-stock.py

# Java
cp templates/Template.java trees/0121-BestTimeToBuyAndSellStock.java
```

Then fill in:
1. **Header** — problem name, link, number, topic, difficulty
2. **Problem Statement** — copy from LeetCode
3. **Examples** — from the problem page
4. **Constraints** — from the problem page
5. **Approach** — your explanation in plain English
6. **Complexity** — time and space
7. **Solution code**
8. **Tests** — at least the provided examples + one edge case

## Adding a new language

Create `templates/template.<ext>` following the same header format.
Update the "Languages" badge in the root README.
